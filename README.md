# Memory
Find all pairs and win!

**Server requirements:**
- spigot 1.20.1
- java 1.17
- **ProtocolLib**

**Language**: German

**Installation (plug and play):**
1. Put the .jar file into your plugins folder
2. Run your server -> **done**

**Functions:**
- Play casual memory with your friends
